# Add Rules and Rulesets Together

- allows you to add rules and rulesets into larger rulesets. This can be
  useful if you want to create a ruleset for a dataset out of checks for
  other datasets.

## Usage

``` r
datavarifyr_plus(a, b)

# S3 method for class 'ruleset'
a + b

# S3 method for class 'rule'
a + b
```

## Arguments

- a:

  the first ruleset you wish to add

- b:

  the second ruleset you wish to add
